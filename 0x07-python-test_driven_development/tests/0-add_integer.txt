The 0-add_integer module
======================

Using ``add_integer``
-------------------

First import
``add_integer`` from the ``example`` module:

    >>> add_integer = __import__('0-add_integer').add_integer

Now use it:

    >>>  add_integer(1, 2)
    3

    >>> add_integer(3, 13.5)
    16

    >>> add_integer(-1, -5)
    -6

    >>> add_integer(-1, 5)
    4

    >>> add_integer(-1.4, 5)
    4

>>> add_integer(1, "Hello")
    
    
>>> add_integer("Hello", "World")
